Programming Concepts Documentation

Variables

Variables are used to store data that can be manipulated and retrieved later. In JavaScript, variables are declared using var, let, or const.

// JavaScript
var name = "John";
let age = 30;
const isStudent = true;

// Python
name = "John"
age = 30
is_student = True

// Java
String name = "John";
int age = 30;
boolean isStudent = true;

Data Types

JavaScript supports various data types including primitive types and objects.

  • Primitive Types: String, Number, Boolean, Null, Undefined, Symbol, BigInt
  • Objects: Arrays, Functions, Date, etc.
// JavaScript
let name = "John"; // String
let age = 30; // Number
let isStudent = true; // Boolean
let score = null; // Null
let address; // Undefined
let id = Symbol('id'); // Symbol
let bigNumber = 9007199254740991n; // BigInt

// Python
name = "John" # String
age = 30 # Integer
is_student = True # Boolean
score = None # None
address = None # Undefined equivalent

// Java
String name = "John";
int age = 30;
boolean isStudent = true;
Integer score = null; // Null

Operators

Operators are used to perform operations on variables and values. JavaScript includes arithmetic, assignment, comparison, logical, and bitwise operators.

// JavaScript
let sum = 10 + 5; // Addition
let product = 10 * 5; // Multiplication
let isEqual = (10 == 10); // Equality

// Python
sum = 10 + 5
product = 10 * 5
is_equal = (10 == 10)

// Java
int sum = 10 + 5;
int product = 10 * 5;
boolean isEqual = (10 == 10);

Control Structures

Control structures manage the flow of the program. Common control structures include conditionals and loops.

// JavaScript
if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Python
if age > 18:
    print("Adult")
else:
    print("Minor")

for i in range(5):
    print(i)

// Java
if (age > 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Functions

Functions are reusable blocks of code that perform a specific task.

// JavaScript
function greet(name) {
    return "Hello, " + name;
}
console.log(greet("John"));

// Python
def greet(name):
    return "Hello, " + name

print(greet("John"))

// Java
public class Main {
    public static void main(String[] args) {
        System.out.println(greet("John"));
    }

    public static String greet(String name) {
        return "Hello, " + name;
    }
}

How an API Works

An API (Application Programming Interface) allows different software applications to communicate with each other. Here's a basic example of how to use an API in JavaScript:

<script>
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));
</script>

In this example, we use the fetch function to make an HTTP request to an API endpoint. The response is then converted to JSON and logged to the console.
